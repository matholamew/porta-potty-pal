# 🚽 PortaPotty Finder

A crowd-sourced application to help users locate the nearest portable toilets, complete with ratings, reviews, and delightfully punny names.

## Features

- 📍 Location-based search for nearby portable toilets
- 🗺️ Interactive map display with markers
- 📱 List view of nearby locations with ratings
- 💩 Fun, randomly generated pun names for new locations
- ⭐ Rating system using poop emojis (1-5)
- 💬 User comments and reviews

## Technical Stack

- Frontend:
  - React Native (for cross-platform mobile support)
  - React Native Maps
  - Expo Location
  - AsyncStorage for local data persistence

- Backend:
  - Firebase (Firestore for database)
  - Firebase Authentication
  - Cloud Functions for name generation
